# Police Knowledge Base System Backend - README更新计划

## 项目分析总结

经过详细分析，该项目是一个**公安专网知识库系统**，具有以下核心特性：

### 技术栈
- **后端框架**: Spring Boot 3.5.7 + Java 17
- **AI能力**: Spring AI 1.0.0-M4 + Spring AI Alibaba (DashScope)
- **认证授权**: Sa-Token 1.38.0 + JWT + Redis
- **ORM框架**: MyBatis-Plus 3.5.5
- **数据库**: MySQL 8.0 + OceanBase SeekDB (向量数据库)
- **API文档**: Knife4j (Swagger 3.0)

### 核心功能模块
1. **认证模块** - 登录/登出、Token认证、权限控制
2. **用户管理** - 用户CRUD、状态管理
3. **知识库管理** - 知识库CRUD、状态管理
4. **文档管理** - 文档CRUD、文件解析、热门/最新文档
5. **RAG智能问答** - 基于向量检索的问答系统
6. **流式对话** - SSE流式输出
7. **文档搜索** - 语义搜索、混合搜索

### 独特特性
- 向量化文档处理（Embedding）
- RAG检索增强生成
- 流式对话（SSE）
- 多格式文件解析（Word/Excel/PDF/TXT/Markdown）
- 混合搜索（向量搜索 + 关键词搜索）
